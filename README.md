# Yandex.Metrica component for Vue.js

Vue.js v 2.x used

## Installation

```
npm i vue-ya-metrica --save
```

## Usage

```
   <vue-ya-metrica counter="123456" enable-hash="true" enable-webvisor="true"></vue-ya-metrica>
```

The `counter` **attribute is required**.
